# ReacTarefas | Gerenciador de Tarefas
Esse é um gerenciador de tarefas feito em React.  

O usuário insere o nome da tarefa e pode ou não inserir uma descrição mais detalhada a respeito dela. Adicionando a tarefa ela é inserida com status de incompleta e aparece nas abas "Todas" e "Abertas", com as opções de marcar como finalizada, deletar ou editar.

Clicando em uma tarefa, é possível ler a descrição detalhada dela em uma página diferente.
